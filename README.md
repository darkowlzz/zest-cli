zest-cli
========

[![NPM](https://nodei.co/npm/zest-cli.png)](https://nodei.co/npm/zest-cli/)

A command line interface for zest.

## To use

  1. Install it:

    ```bash
    $ npm i -g zest-cli
    ```
    This would install zest-cli globally.
    
  2. Use:

    ```bash
    $ zest -s file.zst
    ```
